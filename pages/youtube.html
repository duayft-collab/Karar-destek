YouTube Tracker Kurulum Rehberi

1. Yeni GitHub repo aç: github.com/new → tracker-hub
2. Dosyaları yükle: channels.json, scripts/tracker.py, .github/workflows/tracker.yml
3. channels.json'u düzenle: Takip etmek istediğin YouTube kanal ID'lerini ekle
4. GitHub Secrets ekle: YOUTUBE_API_KEY ve OPENAI_API_KEY
5. Bu sayfada SUMMARIES_URL'i güncelle: tracker repo'nun raw data/summaries.json URL'i
6. Actions → Run workflow ile test et
